# Contributing to conference.matrix.org

Thank you for taking the time to contribute to Matrix!

This is the repository for the conference.matrix.org website for the Matrix Conference.

This repository is set up to deploy the main branch to the timeless domain conference.matrix.org as well as the current year subdomain YYYY.matrix.org.
The sites for previous years are archived to YYYY branches matching the respective subdomain of that year.

## Processes

The processes for development on this repository are inherited from the main matrix.org website repo
and can be found at <https://github.com/matrix-org/matrix.org/blob/main/CONTRIBUTING.md>.
